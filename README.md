# Alpha Zeta Website

This repository holds the code for the official website of Alpha Zeta, a premier co-ed business society at USC. The site showcases information about our society, including member biographies, alumni networks, and how to apply.

## Features

- **Interactive Hero Banner**: Engages visitors immediately with a dynamic introduction to Alpha Zeta.
- **About Us Section**: Provides detailed information about the history and mission of Alpha Zeta.
- **Membership Statistics**: Animated statistics showing membership diversity and reach.
- **Alumni and Careers**: Displays companies our alumni work at and career opportunities.
- **Membership Directory**: Interactive grid featuring current members and their roles.
- **Application Section**: Allows prospective members to apply or get in touch with society representatives.

## Technologies Used

- **Next.js**: The React framework for production.
- **Tailwind CSS**: For styling.
- **Framer Motion**: For animation.
- **React CountUp**: For animated numbers.
- **CoreUI**: For carousel components.
